# Changelog

All notable changes to this project should be documented in this file.

## v1.0.0

Initial `Assessment Platform v1.0` release baseline.

### Included

- Simple single-admin authentication with login, logout, and password change
- Dashboard with lightweight operational overview
- Assessment run management with create, active, submitted, and archived workflows
- Template authoring with drafts, versioning, reusable domains, reusable questions, and managed categories
- Libraries area for categories, domains, and questions
- Teams management
- Results experience with overview, compare, peers, print summary, and chart-based drilldown
- Reports area for latest submitted state across teams and across `team + assessment`
- PostgreSQL + Prisma persistence model
- Installation and database documentation
